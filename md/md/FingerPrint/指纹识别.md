# 指纹识别

## 接口

- 指纹录入过程中退出









## DBus 方法

- 查询org.sailfishos.fingerprint1接口

  ```bash
  dbus-send --system --print-reply --dest=org.sailfishos.fingerprint1 /org/sailfishos/fingerprint1 org.freedesktop.DBus.Introspectable.Introspect
  ```

- 查询当前状态

  ```
   dbus-send --system --type=method_call --print-reply --dest=org.sailfishos.fingerprint1 /org/sailfishos/fingerprint1 org.sailfishos.fingerprint1.GetState
  ```

- 查看当前指纹列表

  ```bash
  dbus-send --system --type=method_call --print-reply --dest=org.sailfishos.fingerprint1 /org/sailfishos/fingerprint1 org.sailfishos.fingerprint1.GetAll
  ```

- 开始识别

  ```bash
  dbus-send --system --type=method_call --print-reply --dest=org.sailfishos.fingerprint1 /org/sailfishos/fingerprint1 org.sailfishos.fingerprint1.Identify
  ```

- 开始录入

  ```bash
  dbus-send --system --type=method_call --print-reply --dest=org.sailfishos.fingerprint1 /org/sailfishos/fingerprint1 org.sailfishos.fingerprint1.Identify.Enroll string:"123"
  ```

- 删除某指纹

  ```bash
  dbus-send --system --type=method_call --print-reply --dest=org.sailfishos.fingerprint1 /org/sailfishos/fingerprint1 org.sailfishos.fingerprint1.Identify.Remove string:"123"
  ```

  

- 监听信号

  ```bash
  dbus-monitor --system interface=org.sailfishos.fingerprint1
  ```

  - 监听状态变化信号

    ```bash
    dbus-monitor --system \ "type='signal',interface='org.sailfishos.fingerprint1',member='StateChanged'"
    ```

  - 监听错误信息

    ```bash
    dbus-monitor --system \ "type='signal',interface='org.sailfishos.fingerprint1',member='ErrorInfo'"
    ```
